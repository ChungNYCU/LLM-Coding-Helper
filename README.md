## Documentation

### Getting start

#### 1. Clone this repository and create `.env` file in the directory like example below.
```
OPENAI_API_KEY = YOUR API KEY
```
 
#### 2. Install dependencies 

##### Windows:
`run setup.bat`

##### Unix:
`run setup.sh`
 
#### 3. Run the project on local environment
`python .\llm_coding_helper.py`
